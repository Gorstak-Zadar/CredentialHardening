# 🛡️ CredentialHardening

> **Credential Security Hardening** - Applies system-wide credential protection policies.

---

## 📋 Overview

CredentialHardening implements comprehensive credential protection including LSA protection, cached credential clearing, and Credential Guard enablement.

---

## 🎯 What It Does

- 🔒 **LSA Protection** - Enable RunAsPPL for LSASS
- 🚫 **Cache Clearing** - Clear cached credentials
- 🛡️ **Credential Guard** - Enable virtualization-based security
- 📝 **Policy Enforcement** - Credential security policies

---

## 🚀 Usage

```powershell
# Run as Administrator
.\CredentialHardening.ps1
```
